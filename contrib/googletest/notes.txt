While in the top of the isce-3.0 repo in my local copy I have created a path “contrib/googletest”.  Under that I put the “include” and “src” directories from the googletest package (that’s the way it comes from googletest):

The following steps install the library and headers in the place where I install isce-3.0 libraries and headers:

> cd $ISCE3_HOME
> ls contrib
googletest
> cd contrib/googletest
>
> ls
include src
>
> g++ -isystem ./include -I./ -pthread -c src/gtest-all.cc
> ar -rv $HOME/tools/lib libgtest.a gtest-all.o
> rm gtest-all.o
>
> cp -R include/gtest /Users/ericmg/tools/include/isce-3.0
>
