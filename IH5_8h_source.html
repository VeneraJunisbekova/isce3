98  // Function to read a dataset from file to memory variable. The input parameter
99  // is a raw point/vector/valarray that will store the (partial) multi-dimensional
100  // dataset.
101  // Input vector/valarray is initialized by caller, but resized (if needed) by the
102  // function. Raw pointer have to be allocated by caller.
103  // Partial extraction of data is possible using startIn, countIn, and strideIn.
104  // startIn: Array containing the position of the first element to read in all
105  // dimension. SizeIn size must be equal to the number of dimension
106  // of the dataset. If nullptr, startIn values are 0.
107  // countIn: Array containing the number of elements to read in all dimension.
108  // countIn size must be equal to the number of dimension of the dataset.
109  // strideIn:Array containing the stride between elements to read in all dimension.
110  // strideIn size must be equal to the number of dimension of the dataset.
111  // If nullptr, stridIn values are 1.
112  //
113  // Examples:
114  // - Dataset contains a 3-bands raster. Dimensions are (100,100,3).
115  // To retrieve the full second band:
116  // startIn=[0,0,1], countIn=[100,100,1], strideIn=nullptr or [1,1,1]
117  // To retrieve the first band, but only every other elements in X direction:
118  // startIn=[0,0,0], countIn=[50,100,1], strideIn=[2,1,1]
