45  // Helper function to map File access options between H5py-type and HDF5 types;

47  switch (mode) {
48  case 'r': return H5F_ACC_RDONLY;

136  // Function to read a dataset from file to memory variable. The input parameter
137  // is a raw point/vector/valarray that will store the (partial) multi-dimensional
138  // dataset.
139  // Input vector/valarray is initialized by caller, but resized (if needed) by the
140  // function. Raw pointer have to be allocated by caller.
141  // Partial extraction of data is possible using startIn, countIn, and strideIn.
142  // startIn: Array containing the position of the first element to read in all
143  // dimension. SizeIn size must be equal to the number of dimension
144  // of the dataset. If nullptr, startIn values are 0.
145  // countIn: Array containing the number of elements to read in all dimension.
146  // countIn size must be equal to the number of dimension of the dataset.
147  // strideIn:Array containing the stride between elements to read in all dimension.
148  // strideIn size must be equal to the number of dimension of the dataset.
149  // If nullptr, stridIn values are 1.
150  //
151  // Examples:
152  // - Dataset contains a 3-bands raster. Dimensions are (100,100,3).
153  // To retrieve the full second band:
154  // startIn=[0,0,1], countIn=[100,100,1], strideIn=nullptr or [1,1,1]
155  // To retrieve the first band, but only every other elements in X direction:
156  // startIn=[0,0,0], countIn=[50,100,1], strideIn=[2,1,1]
