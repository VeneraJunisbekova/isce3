Detailed Description

template<class T>
class isce::signal::NFFT< T >

Non-equispaced fast Fourier transform (NFFT)

Class implementing NFFT algorithm described in [3] . This implementation differs in that the grid points do not need to be specified ahead of time, which is more convenient for SAR backprojection. Typical usage will entail three steps:

  1. Construct NFFT object
  2. Feed it regularly-sampled frequency-domain data with set_spectrum.
  3. Request arbitrary time-domain samples with interp.
