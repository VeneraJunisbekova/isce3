Search function for given name in the group.

Parameters
[in]nameRegular Expression to search for.
[in]startRelative path from current group to start the search from.
[in]typeType of object to search for. Default: BOTH
[in]returnedPathAbsolute or Relative path of found object. Default: FULL

The function returns paths of all objects in the file whose names satisfy name.

param name: can be a regular expression. param type: three types of objects to search for are available: GROUP: only returns groups whose names satisfy the inpout name. DATASET: only return datasets whose names satisfy the input name. BOTH: return groups and datsets whose names satisfy the input name. param returnedPath: the returned path of the found objects can be expressed from the current group (returnedPath = FULL - this is default) or relative to the start (returnedPath = RELATIVE).
